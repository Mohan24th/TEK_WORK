# Online Complaint Management System

## Description
A Streamlit-based application where users submit complaints and admins manage complaint status using MySQL.

## Features
- User complaint submission
- Auto-generated Complaint ID
- Complaint status tracking
- Admin panel to view and update complaints

## Tech Stack
- Python
- Streamlit
- MySQL

# Student Performance Management System

## Description
A Streamlit web application to manage and analyze student performance using MySQL.

## Features
- Add, view, update, delete students
- Pass/Fail status
- Average marks & pass percentage
- Top scorer identification
- Data visualization using charts

## Tech Stack
- Python
- Streamlit
- MySQL
- Pandas
- Matplotlib

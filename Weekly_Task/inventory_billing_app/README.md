# Inventory Management App
- A Streamlit-based application where users manage invntory and billing using MySQL.
# Features
- Add/Delete/Edit/View products
- Generate bill
- Generate invoice
- Download bill in txt format

# Tech Stack
- Pyhton
- Streamlit
- MySql

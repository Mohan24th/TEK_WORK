# Student Attendance & Marks Portal

## Description
A Streamlit-based application to manage student attendance and subject-wise marks using MySQL.

## Features
- Add student details
- Mark daily attendance
- Store subject-wise marks
- Attendance history
- Attendance percentage calculation
- Pass/Fail status

## Tech Stack
- Python
- Streamlit
- MySQL
- Pandas
